## SYNOPSIS

Returns meta data about all the tasks defined in the provided psake script.

## SYNTAX

```powershell
Get-PSakeScriptTasks [[-buildFile] <String>] [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION

Returns meta data about all the tasks defined in the provided psake script.
